## ğŸ” Research Process Per City

### Step 1: Preliminary Research (10 min)
- [ ] Check if city already in database
- [ ] Verify population and contact info
- [ ] Identify jurisdiction hierarchy

### Step 2: Regulations (25 min)
- [ ] Download city building/plumbing code
- [ ] Check for greywater-specific sections
- [ ] Verify state code adoption
- [ ] Document allowances and restrictions
- [ ] Save PDFs and note URLs

### Step 3: Permits (20 min)
- [ ] Access permit portal
- [ ] Get fee schedule
- [ ] Document requirements
- [ ] Note processing times
- [ ] Test application URLs

### Step 4: Incentives (15 min)
- [ ] Check city water utility rebates
- [ ] Check county programs
- [ ] Check water district programs
- [ ] Document amounts and eligibility

### Step 5: Data Entry (10 min)
- [ ] Fill out city data template
- [ ] Run validation checks
- [ ] Insert into database
- [ ] Update research tracking
